### Angular UI Bootstrap 練習題

將 angular ui bootstrap 的 tabset 組件加上雙擊即移除的事件處理

### 提供功能

- double click tab: 移除 tab 和對應內容
- 點擊 tab 右方的 x 圖示: 移除 tab 和對應內容
- need_remove: boolean，true 表示加上移除功能，false 表示不加上移除功能
